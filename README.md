### Prerequisites

- **Node.js**: Latest version (tested with Node.js **20+**).
- **Angular CLI**: Latest version globally installed (`npm install -g @angular/cli`). Uses Angular CLI **19+**.

### Steps to Run

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm start
   ```
4. Run unit tests:
   ```bash
   npm test
   ```
5. Run linting:
   ```bash
   npm run lint
   ```
